何謂 Canvas？

Canvas 是 HTML tag 的一種，就像是在網頁裡開一塊空白的畫布，利用 JavaScript 在畫布上繪圖。透過 Canvas 還可以做出一些簡單的動畫，再加上事件監聽完成有趣的前端互動、小遊戲。
